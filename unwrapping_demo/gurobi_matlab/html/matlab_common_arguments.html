Common Arguments

Most common arguments in the Gurobi MATLAB interface are MATLAB struct variables, each containing multiple fields. Several of these fields are optional. Note that you refer to a field of a struct variable by adding a period to the end of the variable name, followed by the name of the field. For example, model.A refers to field A of variable model.
